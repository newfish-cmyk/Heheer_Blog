# Heheer_Blog

一个前端初学者的前后端练手项目，结构简单，逻辑清晰，适合练习。

## 🍭 Tech Stack
- 前台：Next.js + Tailwind

- 中台：egg.js

- 后台：React hooks + antd

- 其他：marked + highlight.js(Markdown解析) & ESLint + Prettier(代码格式化)
